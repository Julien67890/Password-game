## 🚀 INSTALLATION

### Étape 1 : Télécharger les fichiers

Téléchargez ces 6 fichiers dans un même dossier :
- index.html
- script.js
- style.css
- manifest.json
- sw.js
- icon.jpg

### Étape 2 : Créer les icônes PNG

1. Convertissez **icon.jpg** en PNG
2. Redimensionnez en 192×192 px → **icon-192.png**
3. Redimensionnez en 512×512 px → **icon-512.png**
4. Placez les 2 PNG dans le même dossier

### Étape 3 : Ouvrir le jeu

1. Double-cliquez sur **index.html**
2. Ou hébergez sur un serveur web
3. Jouez !

## 📱 INSTALLATION MOBILE (PWA)

### Sur iOS (iPhone/iPad)

1. Ouvrez **index.html** dans Safari
2. Appuyez sur l'icône Partager 📤
3. Sélectionnez "Sur l'écran d'accueil"
4. Nommez l'app et appuyez sur "Ajouter"
5. ✅ L'icône apparaît sur votre écran !

### Sur Android (Chrome)

1. Ouvrez **index.html** dans Chrome
2. Menu ⋮ → "Ajouter à l'écran d'accueil"
3. Ou une bannière apparaîtra automatiquement
4. ✅ L'app est installée !

## 🔧 RÉSOLUTION DES PROBLÈMES

### Le mode vocal ne fonctionne pas

✅ **Solution** :
- Utilisez Chrome ou Edge (meilleure compatibilité)
- Vérifiez les autorisations micro
- Le badge doit afficher "✓ supporté"

### Message "Groq indisponible"

✅ **Solution** :
- Videz le cache : CTRL+SHIFT+DELETE
- Ou mode navigation privée : CTRL+SHIFT+N
- Ou forcez le rechargement : CTRL+F5

### L'installation PWA n'apparaît pas

✅ **Solution** :
- Vérifiez que icon-192.png et icon-512.png existent
- Utilisez HTTPS ou localhost
- Rechargez la page
