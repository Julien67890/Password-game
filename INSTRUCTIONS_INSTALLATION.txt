🚀 INSTALLATION RAPIDE:
═══════════════════════════════════════════════════════════════

ÉTAPE 1: Créer les icônes PNG
------------------------------
1. Allez sur https://www.iloveimg.com/fr/convertir-en-png
2. Uploadez icon.jpg
3. Convertissez en PNG
4. Redimensionnez en 192×192px → Sauvez comme "icon-192.png"
5. Redimensionnez en 512×512px → Sauvez comme "icon-512.png"
6. Placez les 2 PNG dans le même dossier que les autres fichiers

ÉTAPE 2: Ouvrir le jeu
----------------------
1. Double-cliquez sur index.html
   OU
2. Faites clic droit → "Ouvrir avec" → Chrome/Firefox/Edge

ÉTAPE 3: Vider le cache (IMPORTANT!)
------------------------------------
1. Appuyez sur CTRL + SHIFT + DELETE
2. Cochez "Images et fichiers en cache"
3. Cliquez sur "Effacer les données"
4. Rechargez la page (F5)

═══════════════════════════════════════════════════════════════
📱 INSTALLATION MOBILE (PWA):
═══════════════════════════════════════════════════════════════

SUR iOS (iPhone/iPad):
----------------------
1. Ouvrez index.html dans Safari
2. Appuyez sur le bouton Partager 📤
3. Sélectionnez "Sur l'écran d'accueil"
4. Nommez l'app "Mot de Passe"
5. Appuyez sur "Ajouter"
✅ L'icône apparaît sur votre écran d'accueil!

SUR ANDROID:
-----------
1. Ouvrez index.html dans Chrome
2. Menu ⋮ (3 points) → "Ajouter à l'écran d'accueil"
3. OU une bannière "Installer" apparaîtra automatiquement
✅ L'app est installée!

🐛 RÉSOLUTION DES PROBLÈMES:
═══════════════════════════════════════════════════════════════

PROBLÈME: Les thèmes ne s'affichent pas
SOLUTION: Videz le cache (CTRL+SHIFT+DELETE) et rechargez

PROBLÈME: Le jeu se termine immédiatement
SOLUTION: Vérifiez que script.js fait bien 140+ KB (pas l'ancien)

PROBLÈME: Mode vocal ne fonctionne pas
SOLUTION: 
  - Utilisez Chrome ou Edge (meilleur support)
  - Autorisez l'accès au micro
  - Badge doit indiquer "✓ supporté"

PROBLÈME: L'installation PWA n'apparaît pas
SOLUTION:
  - Vérifiez que icon-192.png et icon-512.png existent
  - Utilisez HTTPS ou localhost
  - Rechargez la page
